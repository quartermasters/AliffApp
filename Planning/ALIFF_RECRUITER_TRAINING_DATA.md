**Innovation**: AI-conducted conversational interviews replace traditional phone screens

- Role-specific questions (e.g., proposal writers get RFP scenarios)
- Adaptive follow-ups (AI probes based on answers)
- Empathetic tone (not interrogation, friendly conversation)
- Immediate feedback (candidate knows next steps within minutes)

## Technical Implementation Notes

### Multi-AI Consensus Scoring
**Why**: Prevents single-model bias, increases evaluation reliability

**How**:
1. Skills test submission received
2. Sent to 3 models simultaneously:
   - GPT-4 Turbo (general intelligence, writing quality)
   - Claude 3.5 Sonnet (compliance, structured thinking)
   - Gemini 1.5 Pro (technical accuracy, code review)
3. Each scores 0-100 with detailed feedback
4. Consensus = average of 3 scores
5. If spread >20 points (e.g., GPT: 90, Claude: 85, Gemini: 65), flag for human review

**Example Output**:
```json
{
  "gpt4_score": 88,
  "gpt4_feedback": "Excellent writing quality, clear structure, minor compliance gap in Section L",
  "claude_score": 84,
  "claude_feedback": "Strong technical approach, missed one M requirement, good past performance narrative",
  "gemini_score": 86,
  "gemini_feedback": "Accurate technical details, solid pricing justification, could be more concise",
  "consensus_score": 86,
  "recommendation": "HIRE - strong candidate with minor areas for coaching"
}
```

---

### Semantic Search with Pinecone
**Why**: Match providers to projects by meaning, not just keywords

**Example**:
- **Query**: "Find proposal writer with VA healthcare experience"
- **Traditional Keyword Search**: Looks for exact words "VA" and "healthcare"
  - Misses: Candidate who wrote "Veterans Affairs medical systems" (different words, same meaning)
- **Semantic Search**: Embeds query, searches by similarity
  - Finds: All candidates with VA/veterans/healthcare/medical experience, regardless of wording

**Implementation**:
1. Embed provider profiles (skills, experience, domains) using OpenAI `text-embedding-3-small`
2. Store in Pinecone with metadata (name, rate, availability)
3. Query time: Embed search request, find top K similar profiles
4. Return ranked results with match scores
